# 扫描历史显示项目名称

## 目标

仅在扫描历史页面（`/runs`）将所属项目从项目 ID 改为项目名称，同时保留项目详情链接。

## 设计

`runs` 处理器在读取最近 100 条扫描记录后，读取现有项目列表并构建 `projectID -> projectName` 映射，将映射和扫描记录一起传给 `runs.html`。

模板使用项目 ID 生成 `/projects/{id}` 链接，但链接文字使用映射中的项目名称。没有项目 ID 的记录继续显示 `—`；ID 不能关联到现有项目的历史记录显示“已删除项目”，避免向用户暴露不直观的 ID。

## 范围与验证

不修改数据库结构、扫描记录模型或其他页面。更新 `/runs` 的页面测试：创建 `p1` / `Local Lab` 项目和关联扫描记录，断言页面显示项目名并保留 `/projects/p1` 链接。
