# 04 — 现代化项目与扫描流程

**What to build:** 统一项目列表、项目创建/编辑、项目详情和扫描创建页面，使配置与主操作形成清晰、紧凑的桌面工作流。

**Blocked by:** 03 — 现代化 Web 扫描报告。

**Status:** done

**Execution skills:** `frontend-visual-design`、`tdd`（仅表单行为 seam）、`browser:control-in-app-browser`、`ponytail`。

- [ ] 项目列表突出名称、范围、默认档位和相关操作，不使用同权重卡片网格堆叠。
- [ ] 项目表单使用统一字段分组、帮助文字、校验和主/次操作层级。
- [ ] 项目详情按概览、扫描配置和历史任务组织，保留所有现有信息和链接。
- [ ] 扫描创建表单保留端口、排除项、档位和高级参数能力，危险或高影响动作有清晰提示。
- [ ] 现有字段名、默认值、提交地址和后端校验保持不变。
- [ ] 相关 Go/JS 测试通过。
- [ ] 提供 1440px 项目列表、项目详情和扫描表单截图，经用户批准后完成。
